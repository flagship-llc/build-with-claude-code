# サンプルプロジェクト テンプレート集

プレ開発期間や当日に使える、様々なツールのアイデアとスタート用プロンプトです。

興味のあるものを選んで、Claude Codeと一緒に作ってみましょう。

---

## 使い方

1. 下記から作りたいものを選ぶ
2. 「最初のプロンプト」をコピーしてClaude Codeに貼り付ける
3. 動いたら「拡張アイデア」を試す

---

## テキスト処理系

### 1. メール下書き生成ツール

**概要**: 要件を入力すると、ビジネスメールの下書きを生成

**想定職種**: 営業、CS、全般

**最初のプロンプト**:
```
メールの下書きを生成するPythonスクリプトを作って。

コマンドラインで以下を入力として受け取る：
- 宛先の名前
- 用件（簡潔に）
- トーン（丁寧/カジュアル）

出力として、件名と本文を生成して表示する。
テンプレートベースでOK、AIは使わなくていい。
```

**拡張アイデア**:
- 複数のテンプレート（お礼、依頼、お詫び等）を選べるように
- 生成結果をクリップボードにコピー
- WebUIを付ける

---

### 2. 日報テンプレート生成

**概要**: 質問に答えていくと日報が完成

**想定職種**: 全般

**最初のプロンプト**:
```
対話形式で日報を作成するPythonスクリプトを作って。

以下の質問を順番に表示して、回答を入力してもらう：
1. 今日やったこと（複数入力可、空エンターで終了）
2. 明日やること
3. 困っていること（あれば）

最後に、フォーマットされた日報を出力する。
```

**拡張アイデア**:
- Markdown形式で出力
- ファイルに保存（日付付きファイル名）
- 過去の日報を参照して継続性をチェック

---

## データ処理系

### 3. CSV整形ツール

**概要**: CSVファイルのクリーニング、フォーマット変換

**想定職種**: マーケティング、経理、コーポレート

**最初のプロンプト**:
```
CSVファイルを整形するPythonスクリプトを作って。

機能：
- 空白行の削除
- 前後の空白をトリム
- 重複行の削除
- 文字コードをUTF-8に統一

使い方: python clean_csv.py input.csv output.csv
```

**拡張アイデア**:
- 特定の列だけ抽出
- 日付フォーマットの統一
- 金額のカンマ区切り追加

---

### 4. 経費科目判定ツール

**概要**: 経費の明細から勘定科目を推定

**想定職種**: 経理、コーポレート

**最初のプロンプト**:
```
経費の勘定科目を判定するPythonスクリプトを作って。

入力: 経費の内容（例: "タクシー代"、"書籍購入"、"会議室利用料"）
出力: 推定される勘定科目（交通費、消耗品費、会議費など）

キーワードベースのルールで判定する方式で。
よく使う科目とキーワードの対応表を内蔵して。
```

**拡張アイデア**:
- CSVで一括判定
- 判定結果の信頼度を表示
- ルールを外部ファイルで管理

---

## ファイル管理系

### 5. ファイル整理ツール

**概要**: フォルダ内のファイルを種類別に自動整理

**想定職種**: 全般

**最初のプロンプト**:
```
指定フォルダ内のファイルを種類別に整理するPythonスクリプトを作って。

- 画像（jpg, png, gif）→ images/
- ドキュメント（pdf, docx, xlsx）→ documents/
- その他 → others/

使い方: python organize.py ~/Downloads
```

**拡張アイデア**:
- 日付別にも整理
- 整理前にプレビュー表示（dry-run）
- 元に戻す機能

---

### 6. ファイル名一括変換

**概要**: 連番付与、日付追加、置換などを一括実行

**想定職種**: 全般

**最初のプロンプト**:
```
ファイル名を一括変換するPythonスクリプトを作って。

機能:
- プレフィックス追加（例: project_001.txt）
- 連番付与
- スペースをアンダースコアに置換
- 小文字に統一

使い方: python rename.py --prefix "project" --numbering ./target_folder
```

**拡張アイデア**:
- 正規表現での置換
- 変更前後のプレビュー
- 元に戻す機能

---

## Web系

### 7. シンプルなWebフォーム

**概要**: 入力フォームとデータ表示のWebページ

**想定職種**: 全般

**最初のプロンプト**:
```
シンプルなアンケートフォームを作って。HTML、CSS、JavaScriptで。

入力項目：
- 名前
- メールアドレス
- 評価（5段階のラジオボタン）
- コメント（テキストエリア）

送信ボタンを押すと、入力内容を画面に表示する（サーバーは不要）。
見た目もきれいにして。
```

**拡張アイデア**:
- 入力バリデーション
- LocalStorageに保存
- CSVでエクスポート

---

### 8. Markdownプレビューア

**概要**: Markdownを入力するとリアルタイムでプレビュー

**想定職種**: 全般

**最初のプロンプト**:
```
Markdownエディタを作って。HTML、CSS、JavaScriptで。

左半分: Markdownを入力するテキストエリア
右半分: リアルタイムでHTMLプレビュー

marked.jsを使ってOK。
```

**拡張アイデア**:
- シンタックスハイライト
- ファイルのドラッグ&ドロップ読み込み
- PDFエクスポート

## 自動化系

### 9. リマインダーBot（ターミナル版）

**概要**: 指定時刻に通知を表示

**想定職種**: 全般

**最初のプロンプト**:
```
シンプルなリマインダーを作って。Pythonで。

使い方: python remind.py "会議の準備" 14:30

指定時刻になったら、macOSの通知で「会議の準備」と表示する。
```

**拡張アイデア**:
- 繰り返し設定（毎日、毎週）
- 複数リマインダーの管理
- Slack通知

---

### 10. 定型作業の自動化スクリプト

**概要**: 毎日やる作業を1コマンドで実行

**想定職種**: 全般

**最初のプロンプト**:
```
朝の定型作業を自動化するPythonスクリプトを作って。

実行すると以下を順番に行う：
1. 今日の日付を表示
2. 天気予報のURL（東京）をブラウザで開く
3. カレンダーアプリを開く
4. 「おはようございます。今日も頑張りましょう！」と通知

macOS用で。
```

**拡張アイデア**:
- 曜日によって処理を変える
- 設定を外部ファイルで管理
- ログを残す

## 選ぶときのポイント

1. **自分の業務で使えそうなもの** を選ぶと、モチベーションが上がります
2. **シンプルなものから始めて**、徐々に機能を追加していきましょう
3. **うまくいかなくても大丈夫**。Claudeに「エラーが出た」と伝えれば修正してくれます
