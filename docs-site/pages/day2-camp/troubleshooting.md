# トラブルシューティングガイド

よくある問題と解決方法をまとめました。

---

## Claude Code関連

### Claude Codeが起動しない

**症状**: `claude` コマンドを実行しても反応がない、またはエラーが出る

**解決策**:

1. Node.jsがインストールされているか確認
   ```bash
   node --version
   ```
   表示されない場合: `brew install node`

2. Claude Codeを再インストール
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

3. ターミナルを再起動

---

### 認証エラー

**症状**: 「認証が必要です」「ログインしてください」等のメッセージ

**解決策**:

1. 再度認証を実行
   ```bash
   claude
   ```
   ブラウザが開くので、再度ログイン

2. それでもダメな場合、キャッシュをクリア
   ```bash
   rm -rf ~/.claude
   claude
   ```

3. メンターに相談

---

### 応答が遅い・止まる

**症状**: Claudeからの応答がずっと返ってこない

**解決策**:

1. `Ctrl + C` で中断
2. 再度同じ指示を送る
3. 頻発する場合は `/clear` してから再試行

---

### Token制限に達した

**症状**: 「制限に達しました」等のメッセージ

**解決策**:

1. **Maxプランへの切り替え**: メンターに声をかけてください（会社負担で切り替えます）
2. 一時的な対処: `/clear` で履歴をクリアすると、少し余裕ができます

---

## コード実行関連

### 「ファイルが見つかりません」

**症状**: FileNotFoundError、No such file or directory

**解決策**:

1. 現在のディレクトリを確認
   ```
   今どのフォルダにいる？
   ```

2. ファイルの存在を確認
   ```
   ls でファイル一覧を見せて
   ```

3. 正しいパスを指定して再実行

---

### 「権限がありません」

**症状**: Permission denied

**解決策**:

1. 実行権限を付与
   ```
   chmod +x でスクリプトに実行権限を付けて
   ```

2. それでもダメな場合
   ```
   pythonコマンドで直接実行して
   例: python script.py
   ```

---

### 日本語が文字化けする

**症状**: 文字化け、□□□、????

**解決策**:

1. Claudeに依頼
   ```
   文字コードをUTF-8に統一して
   ```

2. ファイル読み込み時にエンコーディングを指定
   ```
   ファイルを読み込むとき encoding='utf-8' を指定して
   ```

---

### モジュールが見つからない

**症状**: ModuleNotFoundError、No module named 'xxx'

**解決策**:

1. Claudeに依頼
   ```
   必要なモジュールをインストールして
   ```

2. 手動でインストール
   ```bash
   pip install モジュール名
   ```

## VS Code関連

### ターミナルが開かない

**解決策**:

1. メニューから「ターミナル」→「新しいターミナル」
2. ショートカット: `Ctrl + `` （バッククォート）

---

### ファイルの変更が反映されない

**解決策**:

1. ファイルを保存（`Cmd + S`）
2. VS Codeのウィンドウを再読み込み（`Cmd + Shift + P` → 「Reload Window」）

## 開発で困ったとき

### 何から始めればいいかわからない

**対処**:
1. Claudeに聞く
   ```
   ○○を作りたいんだけど、どこから始めればいい？
   ```
2. 小さな部品から作り始める
3. メンターに相談

---

### 複雑になりすぎてわからなくなった

**対処**:
1. `/clear` で履歴をクリア
2. 最小限の機能だけ作り直す
   ```
   シンプルに○○だけできるものを一から作って
   ```
3. 一つずつ機能を追加していく

---

### エラーが消えない

**対処**:
1. エラーメッセージをそのままClaudeに見せる
2. 「別の方法で試して」と言う
3. それでもダメなら:
   ```
   このアプローチは諦めて、別の方法で○○を実現して
   ```
4. メンターに相談

---

### Claudeの提案が理解できない

**対処**:
1. 説明を求める
   ```
   もっと詳しく説明して
   初心者向けに説明して
   ```
2. 部分的に質問
   ```
   この行は何をしている？
   ```
3. メンターに説明を求める

3. **メンターに共有**: 上記を伝える
