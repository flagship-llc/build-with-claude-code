# Claudeの料金体系

Claude Codeを使うためのプランと、使用量の仕組みについて解説します。

公式の料金ページ: https://claude.com/ja-jp/pricing

---

## プラン比較

| プラン | 月額払い | 年額払い | 主な特徴 |
|--------|----------|----------|----------|
| **Pro** | $20/月 | $17/月（$204/年） | 一般的な開発作業に十分な使用量 |
| **Max 5x** | $100/月 | $85/月（$1,020/年） | Pro の約5倍の使用量 |
| **Max 20x** | $200/月 | $170/月（$2,040/年） | Pro の約20倍の使用量 |

※ 年額払いは約15%割引

### Pro プラン

- Claude Code の全機能が利用可能
- 通常の開発作業には十分な使用量
- ワークショップ参加者には主催から提供

### Max プラン

- **Max 5x**: Pro プランの約5倍の使用量
- **Max 20x**: Pro プランの約20倍の使用量
- 長時間の集中開発セッションに最適
- Token 制限に達した場合のアップグレード選択肢

---

## 5時間セッションウィンドウとは

Claude の使用量制限は**5時間のセッションウィンドウ**で管理されています。

### 仕組み

1. 最初のリクエストを送信した時点で5時間の「窓」が開始
2. その窓の間の使用量がカウントされる
3. 5時間後に窓がリセットされ、次のリクエスト時に新しい窓が開始

### 具体例

```
10:00 - 最初のリクエスト送信 → 5時間ウィンドウ開始
10:00〜15:00 - この間の使用量がカウントされる
15:00 - ウィンドウ終了、リセット
15:05 - 次のリクエスト → 新しい5時間ウィンドウ開始（〜20:05）
```

**ポイント**: 深夜0時などの固定時刻ではなく、**あなたが使い始めた時刻から5時間後**にリセットされます。

### 使用量の確認方法

Claude の設定画面で現在の使用状況を確認できます：

https://claude.ai/settings/usage

![使用量の確認画面](/images/settings_usage.png)

- **セッション使用**: 現在の使用率（%）
- **リセット時刻**: 「Today ○○にリセット」と表示される

### 制限に達したときの対処法

1. **リセット時刻まで待つ**: 画面に表示されるリセット時刻まで待つ
2. **作業を分割**: 一度に大きな変更を依頼せず、小さく分ける
3. **Max プランへアップグレード**: 制限が大幅に緩和される

## トークンとは

Claude との会話で消費される「単位」です。

### トークンの目安

| 内容 | トークン数（概算） |
|------|-------------------|
| 日本語1文字 | 約1〜2トークン |
| 英語1単語 | 約1トークン |
| コード1行 | 約5〜15トークン |

### トークンを多く消費するケース

- 長いファイルを読み込む
- 大量のコードを生成する
- 長い会話を続ける（コンテキストが蓄積）

### トークンを節約するコツ

- 必要なファイルだけを読み込むよう指示
- 会話が長くなったら `/clear` でリセット
- 一度に大きな変更を依頼せず、段階的に進める

## ワークショップでの運用

### 基本方針

- **Claude Pro プラン**を参加者全員に提供
- 制限に達した場合は**Max プラン**へアップグレード対応

### 制限に達したら

1. まずメンターに相談
2. 必要に応じて Max プランへアップグレード
3. 待ち時間があれば、設計を考える時間として活用

## 便利なツール: Claude Usage Tracker

使用量を可視化するChrome拡張機能です。

### インストール方法

1. [Claude Usage Tracker](https://github.com/hamed-elfayome/Claude-Usage-Tracker) にアクセス
2. Chrome Web StoreからChrome拡張機能をインストール
3. Claudeを開くと使用量が表示される

### 確認できること

![Claude Usage Tracker](/images/claude-usage-tracker.png)

- 現在の使用量（%表示）
- 5時間セッションの残り
- リセット時刻

Chrome拡張機能として常に表示されるので、Claudeを使っているときにいつでも使用量を確認できます。

---

## まとめ

- 使用量は**5時間のセッションウィンドウ**で管理される（使い始めから5時間後にリセット）
- Pro プランで通常の開発作業は十分
- 制限に達しても、リセット時刻まで待てば回復する
- ワークショップでは制限を気にせず開発できる体制を整えている
