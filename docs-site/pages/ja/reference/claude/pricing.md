# Claudeの料金体系

Claude Codeを使うためのプランと、使用量の仕組みについて解説します。

公式の料金ページ: https://claude.com/ja-jp/pricing

---

## プラン比較

| プラン | 月額料金 | 主な特徴 |
|--------|----------|----------|
| **Pro** | $20/月 | 一般的な開発作業に十分な使用量 |
| **Max** | $100/月または$200/月 | 大規模プロジェクトや集中的な開発向け |

### Pro プラン

- Claude Code の全機能が利用可能
- 通常の開発作業には十分な使用量
- ワークショップ参加者には主催から提供

### Max プラン

- Pro プランの5倍以上の使用量
- 長時間の集中開発セッションに最適
- Token 制限に達した場合のアップグレード選択肢

---

## 5時間ローリングウィンドウとは

Claude の使用量制限は**5時間のローリングウィンドウ**で管理されています。

### 仕組み

1. 過去5時間の使用量が計算される
2. 制限に達すると一時的に使用できなくなる
3. 5時間経過した分から使用量が回復する

### 具体例

```
10:00 - 大量にトークンを使用
11:00 - まだ制限に近い状態
12:00 - 少し回復
...
15:00 - 10:00の使用分が「窓」から外れ、大きく回復
```

### 使用量の確認方法

Claude の設定画面で現在の使用状況を確認できます：

https://claude.ai/settings/usage

![使用量の確認画面](/images/settings_usage.png)

- **セッション使用**: 現在の使用率（%）
- **リセット時刻**: 「Today ○○にリセット」と表示される

### 制限に達したときの対処法

1. **リセット時刻まで待つ**: 画面に表示されるリセット時刻まで待つ
2. **作業を分割**: 一度に大きな変更を依頼せず、小さく分ける
3. **Max プランへアップグレード**: 制限が大幅に緩和される

## トークンとは

Claude との会話で消費される「単位」です。

### トークンの目安

| 内容 | トークン数（概算） |
|------|-------------------|
| 日本語1文字 | 約1〜2トークン |
| 英語1単語 | 約1トークン |
| コード1行 | 約5〜15トークン |

### トークンを多く消費するケース

- 長いファイルを読み込む
- 大量のコードを生成する
- 長い会話を続ける（コンテキストが蓄積）

### トークンを節約するコツ

- 必要なファイルだけを読み込むよう指示
- 会話が長くなったら `/clear` でリセット
- 一度に大きな変更を依頼せず、段階的に進める

## ワークショップでの運用

### 基本方針

- **Claude Pro プラン**を参加者全員に提供
- 制限に達した場合は**Max プラン**へアップグレード対応

### 制限に達したら

1. まずメンターに相談
2. 必要に応じて Max プランへアップグレード
3. 待ち時間があれば、設計を考える時間として活用

---

## まとめ

- 使用量は**5時間のローリングウィンドウ**で管理される
- Pro プランで通常の開発作業は十分
- 制限に達しても、少し待てば回復する
- ワークショップでは制限を気にせず開発できる体制を整えている
