# Build with Claude Code: Zero to One

*A Hands-On Workshop for Everyone*

AIとペアプログラミングして、1日で自分だけのツールを作るワークショップへようこそ！

ゼロから新しいツールを作るもよし、すでに作りかけのアプリを持ち込んで発展させるもよし。ワークショップ後も継続して開発を続けられます。

---

## このワークショップについて

**エンジニアでなくても大丈夫。** プログラミング経験がなくても、Claude Codeと一緒なら自分のアイデアを形にできます。必要なのは「こんなものがあったらいいな」という気持ちだけ。

**オンサイトでしっかりフォロー。** 少人数制で、メンターが常にサポート。つまずいたらすぐに質問でき、疑問を解消しながら進められます。一人で悩む必要はありません。

**この先の選択肢が広がる。** AIとのペアプログラミングは、これからの時代の新しいスキル。この機会に身につけることで、仕事でも趣味でも、自分でツールを作るという選択肢が生まれます。

---

## イベント概要

- **形式**: 1日ワークショップ（10:00 - 18:00）
- **定員**: 8名程度（少人数制）
- **対象**: エンジニアリング経験不問
