# Achievements

Showcasing apps and tools created by workshop participants.

---

## About Listings

Projects created during the workshop can be featured here.

**You decide what to share**:
- Project name only
- With brief description
- With screenshots
- With GitHub repository link
- Real name or nickname

How much you want to share is completely up to you.

---

## Want to Be Featured?

Let a mentor or organizer know after the workshop.

**Information needed**:
- Project name
- Brief description (optional)
- What you want to share (name, screenshots, links, etc.)
