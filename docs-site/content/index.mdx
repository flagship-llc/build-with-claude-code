# Build with Claude Code: Zero to One

*A Hands-On Workshop for Everyone*

AIとペアプログラミングして、1日で自分だけのツールを作るワークショップへようこそ！

ゼロから新しいツールを作るもよし、すでに作りかけのアプリを持ち込んで発展させるもよし。ワークショップ後も継続して開発を続けられます。

---

## このワークショップについて

**エンジニアでなくても大丈夫。** プログラミング経験がなくても、Claude Codeと一緒なら自分のアイデアを形にできます。伝統的な意味でのエンジニアである必要はありません。必要なのは「こんなものがあったらいいな」という気持ちだけ。それで、エンジニアリングが実現できます。

**オンサイトでしっかりフォロー。** 少人数制で、メンターが常にサポート。つまずいたらすぐに質問でき、疑問を解消しながら進められます。一人で悩む必要はありません。

**この先の選択肢が広がる。** AIとのペアプログラミングは、これからの時代の新しいスキル。この機会に身につけることで、仕事でも趣味でも、自分でツールを作るという選択肢が生まれます。

---

## 作れるもの

基本的には**Webアプリ**を想定していますが、**ネイティブアプリ**やその他のツールも作れます。

| カテゴリ | 例 |
|---------|-----|
| Webアプリ | ダッシュボード、業務管理ツール、フォーム入力ツール、データ可視化 |
| Webサイト | ポートフォリオ、ランディングページ、ブログ |
| スマートフォンアプリ | iOSアプリ、Androidアプリ |
| デスクトップアプリ | macOSアプリ、Windowsアプリ |
| 自動化・Bot | Slack Bot、Chrome拡張機能、定期実行スクリプト |

### Shopifyアプリ

Webアプリの一種として、ECサイト運営者向けのShopifyアプリ開発も可能です。

- 在庫管理・注文管理の拡張
- 顧客分析ダッシュボード
- 独自機能の追加

[Shopify App Store](https://apps.shopify.com/?locale=ja) での展開さえ可能です。なお、Shopify管理画面内のAIアシスタント「Sidekick」でもアプリ作成が可能になっています。

自分の業務や生活で「こんなツールがあったら便利なのに」と思うものを、一緒に作りましょう。

## イベント概要

- **形式**: 1日ワークショップ（10:00 - 18:00）
- **定員**: 8名程度（少人数制）
- **対象**: エンジニアリング経験不問
- **事前準備**: Day 1（セットアップ会）で環境構築を完了してから当日に臨みます。Day 1.5（プレ開発期間）のチュートリアルはオプションです

### 持ち物チェックリスト

- [ ] ノートPC（Mac）
- [ ] 充電器
- [ ] 環境構築済み（事前セットアップ会で完了）
- [ ] 作りたいもののアイデア

### 主催からのサポート

- **メンターサポート**: 環境構築から開発まで、メンターがサポートします
- **Claude Proプラン**: 参加者全員に提供
- **Maxプラン**: Token制限時にアップグレード対応
