# サンプルプロジェクト テンプレート集

プレ開発期間や当日に使える、様々なツールのアイデアとスタート用プロンプトです。

自分の部署・業務に近いものを選んで、Claude Codeと一緒に作ってみましょう。

---

## 使い方

1. 下記から作りたいものを選ぶ
2. 「最初のプロンプト」をコピーしてClaude Codeに貼り付ける
3. 動いたら「拡張アイデア」を試す

## Sales（営業）

### 商談メモ → 議事録変換ツール

**概要**: 箇条書きの商談メモから、フォーマットされた議事録を生成

**最初のプロンプト**:
```
商談メモを議事録に変換するPythonスクリプトを作って。

入力: 箇条書きのメモ（テキストファイル）
出力: 以下のフォーマットの議事録

---
# 商談議事録
日付: [今日の日付]
参加者: [メモから抽出]

## 議題
## 決定事項
## 次回アクション
## 備考
---

使い方: python meeting_to_minutes.py memo.txt
```

**拡張アイデア**:
- Markdown/Word形式で出力
- 顧客名から過去の商談履歴を参照
- Slack/メールで共有
